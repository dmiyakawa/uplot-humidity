# uPlot 湿度・温度デモ

このプロジェクトは、uPlotを使用したリアルタイム時系列データ表示のデモです。
Claude Codeで作成されました。

## 特徴

- Python Flaskバックエンドによる疑似湿度・温度データ生成
- uPlotを使用したリアルタイム折れ線グラフ
- 1秒間隔でのデータ更新
- 開始/停止/クリア機能

## セットアップ

### 1. 依存関係のインストール

```bash
uv pip install -r requirements.txt
```

### 2. サーバーの起動

```bash
uv run python app.py
```

サーバーは `http://localhost:5000` で起動します。

### 3. Webフロントエンドの表示

ブラウザで `http://localhost:5000` にアクセスしてください。

## API エンドポイント

- `GET /api/data` - 現在の時刻の湿度・温度データを返す
- `GET /api/history` - 過去60秒間のデータを返す

## 使用方法

1. バックエンドサーバーを起動
2. ブラウザでHTMLページを開く
3. 「開始」ボタンをクリックしてリアルタイム更新を開始
4. 「停止」でデータ取得を停止
5. 「クリア」でグラフをクリア
