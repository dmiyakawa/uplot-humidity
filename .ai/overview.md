# uPlot + Python デモ

このプロジェクトではuPlotの時系列データ表示のデモを作成する。疑似データ生成にはPythonバックエンドを用いる

Webフロントエンドは単一のHTMLおよび最小構成のJavaScriptとし、フレームワークは使用しない。
HTMLには折れ線グラフと簡易の操作インターフェースがあり、新しいデータが得られたらそれをグラフに足していく、

WebバックエンドはPython 3.13以上を用いて、時系列データを返す単一のRESTエンドポイントを持つ、
バックエンドが返す時系列データは毎秒新たに新しいデータが生成されるものとし、データの中身はグラフの経時の変化がWebフロントエンドで確認できる範囲でランダムのものでよい。
リアルデータである必要はない。
開発用サーバがLinux環境から起動できるようにすること
